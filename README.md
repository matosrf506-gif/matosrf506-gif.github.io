## Features

- **6-language support** — zh / en / vi / ja / ko / tr with per-language navigation and hreflang SEO
- **Hero image preloading** — `<link rel="preload">` on homepage for LCP optimization
- **IntersectionObserver lazy loading** — all post images lazy-load on viewport entry
- **JSON-LD structured data** — Article, FAQPage, HowTo, Organization, Speakable, BreadcrumbList
- **Dark mode** — automatic via `prefers-color-scheme`
- **jekyll-sitemap** — auto-generated sitemap for search engine indexing
- **jekyll-seo-tag** — Open Graph, Twitter Card, canonical URLs out of the box
- **99/100 Lighthouse scores** — performance and SEO optimized

## Tech Stack

- **Static site generator:** Jekyll 4.x
- **Hosting:** GitHub Pages
- **CSS:** Custom properties, responsive grid, glassmorphism header
- **JS:** Vanilla IntersectionObserver (no dependencies)
- **SEO:** jekyll-seo-tag, jekyll-sitemap, jekyll-feed
- **Images:** Pure SVG covers and hero (zero raster images above the fold)
