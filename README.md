## 功能特性

- **6 语言支持** — 中 / 英 / 越 / 日 / 韩 / 土，各语言独立导航与 hreflang SEO
- **首页大图预加载** — `<link rel="preload">` 优化 LCP 首屏性能
- **可视区域懒加载** — IntersectionObserver 按需加载所有文章图片
- **结构化数据** — Article、FAQPage、HowTo、Organization、Speakable、BreadcrumbList
- **深色模式** — 自动跟随系统 `prefers-color-scheme`
- **自动 Sitemap** — jekyll-sitemap 生成，加速搜索引擎收录
- **Lighthouse 99分** — 性能与 SEO 双优化
